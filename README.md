# MovingCursor
This program allows users to move the cursor and click solely with their hands using the camera.

# Usage
- The cursor follows the hand's real-time movement
- The clicking feature occurs when the thumb forms a right angle with the palm.

This is an improved version of Google's Mediapipe Solution regarding hand detection.
Reference: https://github.com/google-ai-edge/mediapipe/blob/master/docs/solutions/hands.md

This script was made on 25th March 2023.
